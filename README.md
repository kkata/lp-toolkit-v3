# lp-toolkit-v3

[Vite](https://ja.vitejs.dev/) で LP を作る環境

HTML は[11ty](https://www.11ty.dev/)、 CSS は[Sass](https://sass-lang.com/)、JavaScript は[TypeScript](https://www.typescriptlang.org/)  
これらを Vite でビルドする

## 使い方

### 1. パッケージをインストール

```bash
npm ci
```

### 2. 開発サーバーを起動

```bash
npm run start
```

### 3. ビルド

```bash
npm run build
```
